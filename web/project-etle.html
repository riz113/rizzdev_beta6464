Miniatur ETLE Berbasis Mikrokontroler

Miniatur Electronic Traffic Law Enforcement (ETLE) adalah proyek yang saya kerjakan sebagai bentuk implementasi sistem penegakan hukum lalu lintas berbasis teknologi. Proyek ini menggunakan mikrokontroler ESP32, Arduino, dan Raspberry Pi untuk mensimulasikan sistem tilang elektronik secara otomatis.

🎯 Tujuan Proyek

Membuat prototipe sistem ETLE yang dapat mendeteksi pelanggaran lalu lintas (misal: menerobos lampu merah) dan mencatatnya secara otomatis menggunakan kamera dan sensor, kemudian mengirimkan data ke server.

⚙️ Teknologi yang Digunakan

ESP32 Arduino Raspberry Pi Kamera Sensor Ultrasonik Python

📌 Fitur Utama

  • Deteksi kendaraan melewati lampu merah menggunakan sensor ultrasonik.
  • Pengambilan gambar oleh kamera yang terhubung ke Raspberry Pi.
  • Penyimpanan data pelanggaran ke database lokal.
  • Notifikasi via LED dan buzzer sebagai penanda pelanggaran.
